Glynn admin

Glynn admin, an admin template build for the love of it.

Paper effects

It is set up using a max width on the container. We like to give a playground to play with, but when monitors get to a 4k monitor resolution it stretched to the max. So we set it up to stech to a max width, but when in max width we added a fading background to create a paper effect on the pages. It is beautifyl in full screen, but scales down responsively to the smallest mobile screen.

Setup, use vh

We learned about a css property vh and found it could be used from as early as IE9. So we did in the page setup. And it looks awesome.
